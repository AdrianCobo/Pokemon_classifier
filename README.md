# Pokemon-classifier

## Install

```
git clone https://github.com/AdrianCobo/Pokemon_classifier.git
cd Pokemon_classifier
unzip images.zip
python3 clasificador.py
```
*Check the imports in the classifier.py file and install the required libraries using pip.

## Run
```
python3 clasificador.py
```
